Algorithm Summaries:

We assume the MLP has output dimension 1 and we denote the input space by \(X\) and the parameter space by \(M\).

An initial parameter \(\theta_0\) is chosen according to the layerwise Gaussian initialization \(W_\ell[a,b]\sim\mathcal N(0,1/n_{\ell-1})\), independently over all weights, with no biases, and a final parameter \(\theta^\star\) is chosen by drawing \(z\sim\mathcal N(0,I_{\dim M})\) and setting \(\theta^\star=\theta_0+z/\|z\|\). These are shared and fixed across all 3 algorithms.

Each algorithm performs batch gradient descent with datasets that are chosen to drive \(\theta\) to \(\theta^\star\). A single batch of the dataset consists of sampled input points \(x_1,\ldots,x_N \in X\) and a target function \(F^\star:X\rightarrow \mathbb R\). The corresponding loss is \(L(\theta)=\frac12\sum_{i=1}^N(F_\theta(x_i)-F^\star(x_i))^2\) and one step of batch gradient descent gives \(\theta_{new} = \theta + \Delta \theta\) where \(\Delta \theta = \eta \sum_{i=1}^N r_i \cdot g_i\), \(r_i = F^\star(x_i) - F_\theta(x_i)\), and \(g_i = \nabla_\theta F_\theta(x_i)\). For each algorithm, we choose \(F^\star\) such that \(\Delta \theta\) is the orthogonal projection of \(w := \theta^\star - \theta\) onto a certain subspace \(V \subset T_\theta M\) that is given as follows:

Gradient span

Set \(V=\operatorname{span}\{g_1,\ldots,g_N\}\). Equivalently, with \(G=[g_1\ \cdots\ g_N]\), choose \(c\) minimizing \(\|Gc-w\|^2\). Then \(\Delta\theta=Gc=\Pi_Vw\).

The corresponding residual is \(r_i=F^\star(x_i)-F_\theta(x_i)=c_i/\eta\), i.e. \(F^\star(x_i)=F_\theta(x_i)+c_i/\eta\).

Hessian span

Let \(H_i=\nabla_x\nabla_\theta F_\theta(x_i):T_{x_i}X\to T_\theta M\), and set \(V=\operatorname{span}\{\operatorname{im}H_i:1\le i\le N\}\). Choose \(v_i\in T_{x_i}X\) minimizing \(\|\sum_iH_i v_i-w\|^2\). Then \(\Delta\theta=\sum_iH_i v_i=\Pi_Vw\).

The corresponding finite-difference dataset uses paired points \(x_i'=x_i+\delta v_i\), with \(\delta>0\) small, and residuals \(r(x_i)=-1/(\eta\delta)\), \(r(x_i')=1/(\eta\delta)\). Equivalently, \(F^\star(x_i)=F_\theta(x_i)-1/(\eta\delta)\) and \(F^\star(x_i')=F_\theta(x_i')+1/(\eta\delta)\). Then \(\eta(r(x_i')\nabla_\theta F_\theta(x_i')+r(x_i)\nabla_\theta F_\theta(x_i))\approx H_i v_i\).

2-Jet span

Set \(V=\operatorname{span}\{g_i,\operatorname{im}H_i:1\le i\le N\}\). Choose \(a_i\in\mathbb R\) and \(v_i\in T_{x_i}X\) minimizing \(\|\sum_i a_i g_i+\sum_iH_i v_i-w\|^2\). Then \(\Delta\theta=\sum_i a_i g_i+\sum_iH_i v_i=\Pi_Vw\).

The corresponding finite-difference dataset uses \(x_i'=x_i+\delta v_i\), with residuals \(r(x_i)=a_i/\eta-1/(\eta\delta)\) and \(r(x_i')=1/(\eta\delta)\). Equivalently, \(F^\star(x_i)=F_\theta(x_i)+a_i/\eta-1/(\eta\delta)\) and \(F^\star(x_i')=F_\theta(x_i')+1/(\eta\delta)\).
